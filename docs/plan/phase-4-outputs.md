# Phase 4 - Outputs and Review Workflow

## Goal

Produce review-friendly outputs for planning decisions and stakeholder discussion.

## Scope

1. Excel workbook export with required sheets.
2. Income Sources + Expenses sheet with person and household totals.
3. Assets sheet including house valuation trajectory.
4. Estate value outputs with taxable estate assumptions.
5. Jupyter notebooks for scenario review, visualization, and interpretation.
6. Charts for assets, taxes, gross/net income, sources of withdrawals, and rates.

## Deliverables

1. Export module and formatting rules.
2. Standard notebook templates that call package modules.
3. Chart generation module for consistent visuals.

## Dependencies

1. Requires stable yearly ledger and optimizer outputs.

## Exit Criteria

1. Workbook columns match planned reporting fields.
2. Exported totals reconcile with internal ledger totals.
3. Notebook workflow can run end-to-end using package API.
