# Phase 6 - Hardening and Maintenance

## Goal

Improve reliability, maintainability, and update readiness.

## Scope

1. Versioned tax parameter tables by year.
2. Extended reconciliation and property tests.
3. Regression and integration test suites.
4. Reproducibility controls for optimization runs.
5. Documentation for assumptions, limitations, and validation status.

## Deliverables

1. Parameter maintenance process.
2. CI-friendly test segmentation.
3. Updated project documentation set.

## Dependencies

1. Requires completion of earlier functional phases.

## Exit Criteria

1. Rule updates can be applied without editing core engine logic.
2. Test coverage catches accounting and tax regressions.
3. Users can understand model limits and trust level from docs.
