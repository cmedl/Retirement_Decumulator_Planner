# Phase 5 - Comparisons and Plan Health

## Goal

Enable decision support across alternative retirement strategies.

## Scope

1. Scenario runner for retirement date comparisons.
2. Return regime comparison (optimistic/base/pessimistic).
3. Inflation stress testing.
4. CPP and OAS start-age comparisons.
5. RRIF conversion and minimum withdrawal behaviors.
6. Annual tax-room indicator.
7. Plan health score tied to objective compliance.

## Deliverables

1. Scenario execution framework.
2. Side-by-side comparison tables.
3. Health diagnostics and warnings.

## Dependencies

1. Requires stable projection API and optimizer output.

## Exit Criteria

1. Changing one scenario input propagates consistently through outputs.
2. Comparison reports show meaningful differences in tax, income, and estate outcomes.
3. Health scoring flags objective misses with clear reasons.
